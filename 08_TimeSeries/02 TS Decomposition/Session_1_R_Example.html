ts() function: - first argumet supplies it the data frame - second argument supplies the start date for the first observation . - third argument identifies the frequency, which in this case is monthly .

The frequency argument lets us specify the number of observations per unit of time. You can specify the number of times that data was collected per year by using the frequency parameter in the ts() function.

For example, if we had monthly data, we would use 12 for the frequency argument, indicating that there are 12 months in the year. While for quarterly time series data you set frequency=4.

set.seed(20)
rv<-rnorm(52) # normal distribution with mean equal to 0 and standard deviation equal to 1

#Annual ==> 1
#Quarterly ==>4
#Monthly ==>12
#Weekly ==> 52

y_Weekly <- ts(rv, start=2003, frequency=52)
y_Monthly <- ts(rv, start=c(2013,3), frequency=12)
y_Quarterly <- ts(rv, start=c(2006,2), frequency=4)
y_Annual <- ts(rv, start=2001, frequency=1)

win.graph(width=12, height=12,pointsize=12)
par(mfrow=c(2,2))
plot(y_Weekly, main="Weekly - White noise",panel.first = grid())
plot(y_Monthly, main="Monthly - White noise",panel.first = grid())
plot(y_Quarterly, main="Quarterly - White noise",panel.first = grid())
plot(y_Annual, main="Annual - White noise",panel.first = grid())

White Noise + Random Walk

White Noise and Random walk Time Series

set.seed(30)  # so you can reproduce the results
NumObs<-150

rv<-rnorm(NumObs) # normal distribution with mean equal to 0 and standard deviation equal to 1

wn <- rv # White Noise
rw <- cumsum(wn) # Random Walk

win.graph(width=12, height=12,pointsize=12)
par(mfrow=c(2,2))
plot(wn, main="White noise",type="l")
plot(rw, main="Random Walk",type="l")
acf(wn, main="ACF - White noise")
acf(rw, main="ACF - Random Walk")
